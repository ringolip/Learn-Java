#### 方法

方法其实就是若干语句的功能集合

参数：进入方法的数据
返回值：从方法中出来的数据

定义方法的完整格式：

```java
修饰符 返回值类型 方法名称(参数名称) {
    方法体;
    return 返回值;
}
```



return两个作用：

- 第一停止当前方法
- 第二将后面的返回值还给调用处

方法的三种调用格式：
1. 单独调用
2. 打印调用
3. 赋值调用

方法三要素：

- 返回值类型
- 方法名称
- 参数列表





##### 方法的重载

多个方法的名称一样，但是参数列表不一样

- 参数数据类型不同
- 参数个数不同
- 参数数据类型顺序不同

与修饰符和返回值类型无关
